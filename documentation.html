scatter plot to show correlation


A pretty powerful tool for analyzing the correlation of the weather conditions has been created. On the left hand-side there is a dashboard that allows the user to customize the scatterplot. The user has the option to choose what will be displayed on the X-axis and the Y-axis with the drop-down lists. There is a constraint here - if a certain parameter is chosen for the X-axis, the same parameter is disabled for the Y-axis and vice versa. This allow the user to choose which parameters he wants see the correlation between. Below the dropdowns, there are radio buttons which allow the user to select the level of the parameters (because in the csv there is data for each parameter as LOW, AVERAGE and HIGH). If the user hovers over the "level*" label, he will see a tooltip explaining that some of the data do not have some of the values (precipitation, wind gust...). Below the radio buttons, there are date pickers that allow the user to choose the exact time interval for the correlation (the Date column from the csv is used). The minimum date is the first date from the csv and the maximum date is the last date of the csv. There are also constraints that prevents the user from choosing a Date from that is larger than Date to, and vice versa. Finally, the user should click on the Show correlation button in order to draw the scatterplot with the chosen parameters. The domains of the axis update and the circles as well. This is all animated for better visual effect. If the user clicks on the button Reset, the default values of all parameters and the scatterplot are displayed. The colors of the circles depend on the event that has happened that particular day. There is a legend on the right for clarification. If two events have happened on a certain day, half of the circle is colored with the color of each event. If 3 events happened on a certain day, there is a gradient to display this. This allows the user to compare multiple events in a single view.
